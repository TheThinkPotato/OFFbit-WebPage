BSEL Mod.

The Tape BSEL Mod. A mod for the core 2 CPUs that changes the FSB clock by tricking the motherboard to think it's CPU runs at a higher FSB. Giving us an overclock, This is a good way to overclock OEM PCs that have no overclocking control in their BIOS.

BSEL CPU Pins

The 775 cpu socket design uses 3 pins to working out the FSB clock speed.

  • BSEL0
  • BSEL1
  • BSEL2

LGA 771 Mod

This mod will enable LGA 771 CPUs into a standard LGA 775 motherboard. This mod requires hardware and bios mods for it to work. See details below about the mod.

Motherboard Chipset Support

The following motherboard chipsets have support for coresponding xeons.

Chipset Core 2 Xeon 5000 series Core 2 Xeon 3000 series
P45, P43, P35, P31, P965 G45, G43, G41, G35, G33, G31 nForce 790i, 780i, 740i, 630i GeForce 9400, 9300 Yes Yes
Q45, Q43, Q35, Q33 X48, X38 Yes Yes
nForce 680i and 650i ? Yes

Known Issues

  • There are issues with this mod and intel built motherboards.
  • Nvidia nForce 680i and 650i don't officialy support 45nm quad cores.
  • Not all Xeon CPUs FSB are supported by motherboards (1600 MT/s either does not boot or only recognises as 1333 MT/s).
  • Make sure CPU TDP is within specification of your motherboard.

LGA 771 Xeon CPUs

  • 3000 series Xeons support 1 CPU where 5000 Series Xeons Support 2 CPUs on a server / workstation motherboard.
  • Most 3000 series Xeons use LGA 775 but some do use LGA 771.
